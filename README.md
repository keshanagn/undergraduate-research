# undergraduate-research
An Investigation to Residual-Based Goodness of Fit Testing for Heteroscedastic Neural Regression Models

The following repository include the necessary code, datasets generated and result tables for simulation of 4 different functions for the undergraduate research. 
1. Sinusoidal regression with constant gaussian errors
2. Sinusoidal regression with constant gaussian mixture errors
3. Detlfsen et al. (2019) function
4. Ishigami function (Pourkamali-Anaraki et al., 2024)

The code is utilized in google colaboratory.
datasets generated are available with csv extension.
Model training will result in result tables which are combined to be excel formats.
